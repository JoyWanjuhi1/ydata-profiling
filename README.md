# ydata-profiling
Exploratory Data Analysis of Diabetes Dataset using Python, Pandas, and ydata-profiling. Includes data cleaning, visualization, and insights for predicting diabetes risk factors.
# Diabetes Data Analysis Project

## Overview
This project explores a diabetes dataset containing medical and demographic information. The goal is to understand the data, identify patterns, and prepare it for potential predictive modeling.

## Tools Used
- Python
- Pandas
- ydata-profiling
- Jupyter Notebook / Google Colab

## Key Steps
- Loaded and explored the dataset using Pandas
- Identified missing and zero values in key medical features
- Performed descriptive statistical analysis
- Generated an automated profiling report using ydata-profiling
- Analyzed correlations between features such as Glucose, BMI, Age, and Outcome

## Key Insights
- Glucose levels show strong correlation with diabetes outcome
- BMI and Age also contribute to diabetes risk
- Dataset contains invalid zero values that likely represent missing data
- Presence of outliers in features like Insulin

## Conclusion
This project demonstrates basic data analysis and preprocessing techniques that are essential for building machine learning models in healthcare datasets.
